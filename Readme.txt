# Disaster Management System

This repository hosts the **Disaster Management System**, a comprehensive project aimed at aiding disaster management processes. It includes features such as weather forecasting, volunteer coordination, and mapping to facilitate efficient disaster response and recovery.

## Features

1. **Weather Forecasting**
   - Integration with APIs to provide accurate weather predictions and alerts.

2. **Quick Emailing**
   - Uses Google APIs to send emergency emails to volunteers and helpers.

3. **Mapping Feature**
   - Displays disaster-affected areas on a map for better situational awareness.

4. **Volunteer Registration Portal**
   - Allows volunteers to register and be part of the disaster relief efforts.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **APIs:** Google APIs, Weather APIs

## Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:AnuraagDev/Disaster-Management-System.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Disaster-Management-System
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

5. Access the application at:
   ```
   http://localhost:3000
   ```

## Usage

1. **Weather Forecasting**:
   - View real-time weather updates and forecasts.

2. **Email Communication**:
   - Quickly send emails to volunteers and emergency contacts.

3. **Mapping**:
   - Visualize affected regions on an interactive map.

4. **Volunteer Portal**:
   - Register as a volunteer and manage disaster relief activities.
